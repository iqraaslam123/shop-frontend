# 🛍️ Shop Frontend

A responsive e-commerce frontend built with **React.js** featuring product browsing, shopping cart functionality, and a modern user interface.

## 🌐 Live Demo

🔗 https://shop-frontend-psi-sandy.vercel.app/

## ✨ Features

- Responsive Design
- Product Listing
- Product Details
- Shopping Cart
- Add & Remove Items
- Quantity Management

## 🛠️ Tech Stack

- React.js
- JavaScript
- React Router DOM
- CSS3
- Vite
